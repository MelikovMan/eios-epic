# Практическое задание 3. для занятий 1,2 блока 2.
## Задание 1: Создайте хук useDebounce
Реализуйте хук, который:

Принимает значение и задержку

Возвращает дебаунсированное значение

Используется для поиска с задержкой

## Задание 2: Создайте хук useGeolocation
Реализуйте хук для получения геолокации:

Возвращает координаты, ошибку и статус загрузки

Поддерживает обновление позиции

Обрабатывает разрешения

## Задание 3: Создайте хук useOnlineStatus
Реализуйте хук для отслеживания онлайн-статуса:

Определяет, онлайн ли пользователь

Слушает события online/offline

Возвращает текущий статус

## Задание 4: Создайте Suspense-совместимый кэш
Реализуйте кэширующую систему для Suspense, которая:

Поддерживает TTL (время жизни)

Позволяет предзагружать данные

Поддерживает инвалидацию по ключу

## Задание 5: Оптимизируйте приложение с Suspense
Возьмите существующее приложение и:

Разделите код с помощью React.lazy

Добавьте гранулярные Suspense boundaries

Реализуйте скелетон-загрузчики

Добавьте Error Boundaries

## Задание 6: Создайте кастомный хук useSuspenseQuery
Реализуйте хук, который:

Интегрируется с вашим API

Поддерживает кэширование

Работает с Suspense

Обрабатывает ошибки

## Задание 7: Создайте блог с RSC
Реализуйте систему блога используя:

Серверные компоненты для статей и списков

Клиентские компоненты для комментариев и лайков

Серверные actions для создания комментариев


Указанные компоненты должны быть на единой веб-странице с возможностью перехода по якорю.
