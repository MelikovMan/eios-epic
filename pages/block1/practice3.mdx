# Практическое задание 3 для урока 4 и 5 и 6.

## Задание 1: Счетчик с историей
Создайте компонент счетчика, который:

Отображает текущее значение

Позволяет увеличивать/уменьшать счетчик

Сохраняет историю всех изменений

Позволяет сбросить счетчик и историю

## Задание 2: Todo List с фильтрацией
Создайте улучшенный список задач:

Добавление/удаление задач

Отметка выполнения

Фильтрация (все, активные, выполненные)

Поиск по задачам

Локальное сохранение

## Задание 3: Кастомный хук useToggle
Создайте кастомный хук useToggle, который:

Принимает начальное значение (по умолчанию false)

Возвращает текущее значение и функцию toggle

Может принудительно устанавливать значение

## Задание 4: Компонент с таймером
Создайте компонент, который:

Запускает таймер при монтировании

Отображает прошедшее время в секундах

Останавливает таймер при размонтировании

Имеет кнопки паузы и сброса

## Задание 5: Кастомный хук useWindowSize
Создайте хук, который отслеживает размер окна браузера и возвращает:

Ширину и высоту окна

Обновляет значения при изменении размера окна

Очищает подписку на события при размонтировании

## Задание 6: Компонент поиска с debounce
Создайте компонент поиска, который:

Отправляет запрос к API при вводе текста

Использует debounce для ограничения частоты запросов

Отображает результаты поиска

Отменяет pending-запросы при новом поиске

## Задание 7: Контролируемая форма регистрации
Создайте форму регистрации с полями:

Имя пользователя

Email

Пароль

Подтверждение пароля

Согласие с условиями

Требования:

Валидация в реальном времени

Проверка совпадения паролей

Кнопка отправки активна только при валидной форме

## Задание 8: Неконтролируемый компонент загрузки аватара
Создайте компонент для загрузки аватара:

Предпросмотр изображения

Валидация типа и размера файла

Кроппинг (опционально)

Возможность удалить загруженное изображение

## Задание 9: Гибридная форма настроек
Создайте форму настроек пользователя:

Контролируемые поля: имя, email

Неконтролируемые: аватар, файлы

Кнопка сохранения с подтверждением
